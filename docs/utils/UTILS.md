# Utils

> [Main Readme](../../README.md) > Utils

Various utilities used by the library.

### [Events](./EVENTS.md)
Event pattern for creating events and triggering callbacks.

### [Case](./CASE.md)
Just functions to convert strings between variable naming conventions

### [Storage](./STORAGE.md)
Helper functions that wrap around localStorage to make the syntax a little friendlier.

### [Types](./TYPES.md)
Functions for dealing with types. 

### [General](./GENERAL.md) 
General functions that dont fit in any other category but are useful for Solana tests.
